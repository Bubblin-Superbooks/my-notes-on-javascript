Introduction

JavaScript was created in the year 1995.

It was originally designed to run only in browsers - like the NetScape, IE or Firefox - but has in recent years grown into full fledged programming language that can run both on the server-side and the client.

This puts javascript in a very unique position that developers of no other programming language can claim, nor enjoy: The ability to write code both for front and backend and create apps that are cross-platform and yet efficient. https://auth0.com/blog/a-brief-history-of-javascript/
